Getting Started

1. Install the SDK

npm install nexvora-sdk

2. Initialize in your app

import { Nexvora } from 'nexvora-sdk';

Nexvora.init({
  projectId: 'YOUR_PROJECT_ID',
  enableBugTracking: true
});

3. Plan your Roadmap

Use our intuitive drag-and-drop interface to manage features, track progress, and align your team effortlessly. Integrate your Git workflow natively to track commits directly on task cards.
