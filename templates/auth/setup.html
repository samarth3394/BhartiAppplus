Welcome to Nexvora!

Let's get your workspace and integrations set up in just two steps.

Step 1: Install Bug Tracking SDK

1. Install via NPM

Run the following command in your project terminal to install the Nexvora SDK.

npm install nexvora-sdk

2. Initialize the SDK

Import and initialize Nexvora at the top of your main application file (e.g., index.js or App.jsx).

import { Nexvora } from 'nexvora-sdk'; Nexvora.init({ projectId: 'YOUR_PROJECT_ID', enableBugTracking: true, captureUnhandledRejections: true });

Step 2: Setup Uptime Monitoring

Monitor Your APIs and Websites

Nexvora will automatically ping your endpoints to ensure they are live. You can set this up directly from your Dashboard once you continue.

  • Go to the Uptime section in your Dashboard.
  • Click on + Add Monitor.
  • Enter your API or Website URL (e.g., https://api.yourdomain.com).
  • Set the ping interval (e.g., every 5 minutes).
  • We will notify you immediately via email if your server goes down.

Step 3: Server Infrastructure Setup (Optional)

Track CPU, RAM, and Disk Usage

To monitor your actual server's health (e.g., AWS EC2, VPS), run the following command on your server terminal. It will set up a lightweight background script that sends metrics to Nexvora.

curl -sSL https://nexvora.com/install.sh | bash -s -- --key=YOUR_NEXVORA_KEY

This script runs a simple cron job every minute. You can view the live charts in the Infrastructure tab on your dashboard.
